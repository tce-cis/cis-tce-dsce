NETSYS-2017 - an inaugural edition of the workshop conducted by Department of Telecommunication Engineering on Feb 06th and 07th 2017.

In order to provide an insight of networking concepts and the technological updates to keep our students walk along the recent trends on communication networks. This workshop focuses on 4th semester students to know the basic computer communication network concepts and networking operating systems. Hands-on training program was given on "VMWare platform" to build knowledge on network programming in C on Linux. VMWare Workstation is a hosted hypervisor that runs Windows and Linux operating system. It enables users to set up virtual machines (VMs) on a single physical machine, and use them simultaneously along with the actual machine. Each virtual machine can execute its own operating system, including versions of Microsoft Windows, Linux, BSD, and MS-DOS.
